# PVSZforQuanX

项目简介
受到Qure项目启发，为QuantumultX策略组，搬运植物大战僵尸(PVSZ)游戏素材。


使用说明
可参考Qure项目进行 远程图标 和 本地图标 使用。
由于植物及僵尸命名繁琐，故使用字母和数字进行命名，请参考效果图进行选择
例如：(P)(01)(02) P=植物 Z=僵尸 01=种类 02=序号
